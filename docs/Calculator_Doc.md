# Shun4miTeX Scientific Calculator Support

Constants:
`pi, e`

Operators:
`+ - * / ^ !`

Grouping:
`( )`

Basic functions:
```text
sqrt(x)
root(x, n)
abs(x)
exp(x)
ln(x)
log(x)
log(x, b)
```

Trig:
```text
sin(x)
cos(x)
tan(x)
arcsin(x)
arccos(x)
arctan(x)
```

Angle conversion:
```text
rad(x)
deg(x)
```

> [!TIP]
> Trig functions use radians. For degrees, use `rad(x)` to first convert from degrees into radians, e.g. `/calc sin(rad(30))`

Combinatorics:
```text
choose(n, k)
perm(n, k)
```
