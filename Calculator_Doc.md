# Shun4miTeX Scientific Calculator Support [WIP, Planned V1]

Constants:
`pi, e`

Operators:
`+ - * / ^ !`

Grouping:
`( )`

Basic functions:
```text
sqrt(x)
root(x, n)
abs(x)
exp(x)
ln(x)
log(x)
log(x, b)
```

Trig:
```text
sin(x)
cos(x)
tan(x)
arcsin(x)
arccos(x)
arctan(x)
```

Angle conversion:
```text
rad(x)
deg(x)
```

Combinatorics:
```text
choose(n, k)
perm(n, k)
```
